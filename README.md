# Cecilia’s Homemade Cuisine – Landing Page

Bienvenido al repositorio de la página web de **Cecilia’s Homemade Cuisine**, una landing page promocional para un restaurante que combina un ambiente acogedor con comida casera y variada.

---

## Descripción del proyecto

Esta página fue creada como una presentación online para un restaurante que ofrece hamburguesas, pizzas, pastas y más, con la opción de **ordenar o reservar mesa**.

Está construida para mostrar:

- Secciones de descuentos/promociones.  
- Menú con distintas categorías (burger, pizza, pasta, fries…).  
- Historia del restaurante.  
- Reservación de mesa (“Book A Table”).  
- Testimonios de clientes.  
- Contacto y formulario para sugerencias o preguntas.  
- Información de contacto y horario.

---

## Objetivos

- Dar visibilidad online al restaurante.  
- Permitir que los clientes vean el menú de forma atractiva.  
- Incentivar la reserva de mesa u ordenar online.  
- Transmitir una atmósfera cálida, casera y profesional.  
- Facilitar la interacción con los clientes.

---

## Tecnologías utilizadas

- **HTML**, **CSS** y **Tailwind CSS** para la estructura y estilos.  
- **JavaScript**  para interactividad.  
- **Diseño responsivo** para compatibilidad móvil.  
- **Imágenes optimizadas** para mejor rendimiento.  
- **Formularios** para contacto y reservas.
